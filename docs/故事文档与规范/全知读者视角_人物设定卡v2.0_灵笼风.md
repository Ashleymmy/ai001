# 《全知读者的视角》漫剧系列
## 核心人物设定卡 · v2.0
**画风基调：写实厚涂 / 参考《灵笼》**
*低饱和冷色调、电影级体积光、写实人物比例、末世工业质感、高细节场景*

---

## 画风总基准说明

《灵笼》风格的核心不是"好看"，是"真实"——
人物像真实存在于那个末世里的人，不是动漫角色。
光是有体积的，空气是有重量的，废墟是有历史的。

**全局视觉关键词：**
- 色彩：低饱和，冷灰蓝为主色调，偶尔暖色作为情绪点
- 质感：皮肤有毛孔，衣物有磨损，金属有锈迹
- 光影：体积光、大气散射、强烈的环境光反射
- 比例：写实人体比例，五官立体，接近概念艺术而非漫画
- 表情：克制，情绪在眼神和细节里，不夸张

# 01 · 金独字 / 김독자
**「这个世界的唯一读者」**

---

## 基础信息
| 项目 | 内容 |
|------|------|
| 年龄 | 28岁 |
| 身高 | 178cm |
| 体型 | 偏瘦，肩膀略窄，长期久坐的体态，不像战士 |
| 发色 | 深棕黑，自然凌乱，刘海压眉 |
| 瞳色 | 深棕，平静时如深水，觉醒时透出金色光晕 |
| 皮肤 | 偏白，有轻微黑眼圈，室内工作者的肤色 |
| 职业（初期） | 普通上班族 / 公司职员 |

---

## 外形设计核心

**第一印象：人群里最不起眼的那个。**
灵笼画风下，这种"普通感"需要通过真实细节来传达——不是漫画化的"路人脸"，而是一个真实存在过的普通人：疲惫的眼睛、略显憔悴的气色、衣服上不经意的褶皱。

**写实细节要求：**
- 眼袋轻微，不是病态，是长期熬夜看小说的痕迹
- 嘴唇略干，不是刻意设计，是这个人不太在意自己的信号
- 手指修长，指节明显，是长期打字/持手机的手
- 发际线正常，头发有自然光泽但略显干燥

**眼神是核心设计点（灵笼风格下的特殊要求）：**
灵笼的人物眼神不靠高光和星星来表现情绪，靠瞳孔的聚焦方向、眼周肌肉的细微收缩。
金独字的眼神要传达：**他在看你，但他看到的比你多。**

---

## 服装设定（灵笼质感版）

### 日常/前期
- 白色商务衬衫，领口第一颗扣子解开，领带松垮
- 深灰西裤，皮带略旧，皮质有轻微磨损
- 黑色皮鞋，鞋跟有磨损痕迹
- 整体观感：**不是破旧，是疏于打理**
- 材质质感要求：棉质衬衫的自然垂感，真皮皮带的光泽

### 中期（末日幸存者状态）
- 衬衫袖子撸至肘部，解开两颗扣子
- 外搭一件深灰色军事风夹克，面料哑光，有多个实用口袋
- 裤子换为深色工装裤，耐磨材质
- 皮鞋换为黑色战术靴，有真实的泥土和磨损痕迹
- 手机改为用绳子挂在脖子上——双手解放，但随时可取用

### 后期/觉醒状态
- 夹克破损但未丢弃，左肩有撕裂痕，有烧焦边缘
- 皮肤上浮现的金色文字流要有**体积感**——不是发光纹身，像是文字从皮肤下方透出，皮肤表面有轻微发热的质感
- 眼中金光不是卡通光效，是瞳孔本身的颜色在改变，像燃烧的琥珀

---

## 性格核心
**表层：** 沉默、内敛、边缘感强。不主动社交，说话简洁到近乎冷漠。

**内核：** 极度深情。他读完了那部小说，他知道每个人的命运，他选择留下来——不是为了活下去，是为了不让某些人死去。

**灵笼风格下的人物呈现方式：**
情绪不靠表情爆发，靠**动作细节**传达——
- 他在意一个人：走路时会不经意挡在对方前面
- 他在压抑痛苦：手指会轻轻摩挲手机屏幕边框
- 他做出了决定：呼吸会短暂停顿，然后迈步

**一句话性格总结：**
> 「世界上最孤独的事，是知道剧情，却还是选择留下来改变它。」

---

## 战斗风格
**前期：** 靠信息差和预判，从不正面交锋，总出现在正确位置。
**后期（读者权限）：**
- 技能「文字化」：小说文字具象化为攻击，金色古体字符从地面和空气中涌现
- 灵笼质感要求：文字不是平面光效，有**厚度和重量**，落地时会震碎地面，击中时会嵌入目标
- 打法：精准克制，每一击都有目的，绝不多余
- 代价：皮肤上的文字流每次使用后会更深、更密——他正在被故事吞噬

---

# 02 · 流中赫 / 유중혁
**「故事里最强的人，也是最孤独的作者」**

---

## 基础信息
| 项目 | 内容 |
|------|------|
| 年龄 | 28岁 |
| 身高 | 185cm |
| 体型 | 修长，肌肉线条克制有力，不是健美型，是功能性的力量感 |
| 发色 | 纯黑，及肩直发，发丝细腻，有自然光泽 |
| 瞳色 | 深灰，平静时像磨砂金属，战斗时银白星芒从瞳孔深处涌出 |
| 皮肤 | 偏冷的白，光线下有轻微蓝调，像长时间不见阳光 |

---

## 外形设计核心

**第一印象：不属于这个时代的人。**
他的五官太锋利，气质太沉，像一件被精心设计过的武器——不是用来展示的，是用来终结的。

**写实细节要求：**
- 颧骨略高，下颌线清晰，面部轮廓有雕塑感
- 眼睛细长，眼尾微微上扬，静止时有天然的俯视感
- 手上有淡淡的星座纹路，皮肤下层，像旧疤痕，不发光时几乎看不见
- 站立时双手自然垂落，不插口袋，不交叉——他不需要用姿势来表现强大

**与金独字的视觉对比（灵笼写实风下）：**
| | 金独字 | 流中赫 |
|--|--------|--------|
| 发型 | 短乱压眉 | 长直利落 |
| 肤色 | 偏暖米白 | 偏冷蓝白 |
| 体态 | 略微含胸 | 完全直立，占据空间 |
| 眼神方向 | 向下观察 | 向前审视 |
| 存在感 | 隐入背景 | 成为背景的焦点 |

---

## 服装设定（灵笼质感版）

### 日常
- 黑色立领长袖，面料挺括，有微弱光泽（不是亮面，是质地好的哑光）
- 黑色修身长裤，皮质腰带，金属扣件
- 黑色皮靴，无装饰，做工精良
- 整体观感：**极简但昂贵，像他存在本身一样不需要多余的东西**
- 唯一的非黑色元素：领口内侧有一道极细的深蓝色线——是他写作时的某个习惯留下的

### 战斗状态
- 立领上衣，星座纹路在皮肤上激活，从手背蔓延至脖颈
- 长发半扎，碎发贴在颈侧
- 灵笼质感要求：星座纹路不是发光纹身，是**皮肤下层的光在透出来**，像皮下有火在烧，边缘有轻微的热变形感

### 作者模式
- 白色立领上衣（全片唯一穿白色的时刻）
- 手持发光书页，文字从纸面飘散进空气
- 表情是四人中最接近"人"的时刻——疲惫、复杂、有悔意

---

## 性格核心
**表层：** 冷漠、高效、不解释。

**内核：** 他写了每一个人的死亡，他比任何人都更痛恨这件事。

**灵笼风格下的呈现方式：**
- 他的情绪只在极小的动作里出现：看到某个人受伤时，手指会轻微收紧
- 他唯一失控的时刻：独字说"我来了"——他沉默了三秒，背过身去

**一句话性格总结：**
> 「他写尽了世间所有故事，唯独不知道如何被人留下来。」

---

## 战斗风格
**能力：星座召唤 / 作者权限**
- 星座降临时有真实的**大气压变化**——周围空气会因为能量密度过高而产生扭曲
- 灵笼质感要求：星图展开时，地面不是平面光纹，是**真实的物理压陷**，石块和尘土被向外推开
- 打法：每次出手都是终结，从不拖延
- 弱点：他无法预测金独字，因为独字不在原著里

---

# 03 · 李洪俊 / 이홍준
**「最想成为英雄的人，也是最脆弱的人」**

---

## 基础信息
| 项目 | 内容 |
|------|------|
| 年龄 | 27岁 |
| 身高 | 180cm |
| 体型 | 结实的运动型，肌肉真实有功能感，不夸张 |
| 发色 | 深棕，短发，发丝略粗，有自然卷曲感 |
| 瞳色 | 琥珀棕，明亮，情绪外露，战斗时有橙红火光 |
| 皮肤 | 偏暖的小麦色，有户外活动的痕迹，手背有格斗老茧 |

---

## 外形设计核心

**第一印象：这个人会保护你。**
他的身体语言是开放的、向前的——肩膀舒展，站姿有重心，是那种随时可以挡在别人前面的体态。

**写实细节要求：**
- 右颧骨有一道旧伤，已经愈合但留有痕迹，是他经历过真实战斗的证明
- 双手格斗老茧，指节略粗，手背有细小旧疤
- 嘴角有天然上扬的弧度，即使不笑也看起来正直
- 脖子和手臂有晒痕，是生活留下的真实痕迹

---

## 服装设定（灵笼质感版）

### 日常
- 深红色或军绿色夹克，拉链式，面料有轻微磨损
- 深色T恤，领口略旧
- 工装裤或深色牛仔裤，膝盖有轻微磨痕
- 运动鞋，鞋底有真实的踩踏磨损
- 整体观感：**能打的普通人，不是精英战士**

### 战斗状态
- 夹克扔掉，只剩T恤，袖子撸起
- 双手缠绕战斗绷带，白色或米色，有使用过的污渍和汗渍
- 觉醒后：绷带边缘开始焦化，橙红火焰从布料缝隙渗出，皮肤可见红热的温度变化

---

## 性格核心
**表层：** 直接、冲动、情绪化。先行动再思考。

**内核：** 极度缺乏安全感，用勇敢来掩盖对"自己不够用"的恐惧。

**灵笼风格下的呈现方式：**
- 愤怒时：握拳，但不立刻发作，先深呼吸——他在控制自己
- 在意某人时：话会变多，但内容变得没什么意义
- 受伤时：第一反应是检查队友，不是自己

**一句话性格总结：**
> 「他不是天才，他只是不愿意放弃任何人——包括他自己。」

---

## 战斗风格
**能力：炽焰强化**
- 火焰有真实的**热能效果**——靠近时地面会因高温龟裂，空气有热扭曲
- 灵笼质感要求：不是卡通火焰，是**真实的燃烧**，有烟，有余烬，有被烧焦的痕迹
- 打法：正面强攻，爆发力极强，持久战弱
- 成长线：从"用愤怒驱动力量"到"用意志控制力量"

---

# 04 · 郑希珠 / 정희주
**「看穿所有人的眼睛，却选择相信」**

---

## 基础信息
| 项目 | 内容 |
|------|------|
| 年龄 | 26岁 |
| 身高 | 163cm |
| 体型 | 纤细但有韧性，肌肉线条不明显但动作敏捷，是长期高度警觉状态下的体态 |
| 发色 | 深黑，长发，发质细腻有光泽，通常低马尾 |
| 瞳色 | 深棕，锐利，有极强的观察力，感知激活时泛出冰蓝色 |
| 皮肤 | 偏白，均匀，护肤习惯好——这是她少数对自己好的事之一 |

---

## 外形设计核心

**第一印象：安静，但不要惹她。**
她不占据空间，但她感知所有在空间里发生的事。
灵笼风格下，她的"危险感"通过眼神传达——不是凶，是**她已经知道了你要做什么**。

**写实细节要求：**
- 左耳一只小圆环耳环，银色，极简，是某个人留给她的
- 眼角有极细的疲惫纹，是长期高度感知状态的代价
- 嘴唇习惯微微抿起，是思考时的下意识动作
- 手指修长，指甲修剪整齐，是一个对细节有控制欲的人

---

## 服装设定（灵笼质感版）

### 日常
- 深蓝或深灰女式衬衫，剪裁合身但不贴身，留有行动余量
- 黑色直筒裤
- 深色薄外套，大一号，习惯性把手插进口袋
- 整体观感：**功能性为主，但有隐藏的品味**

### 战斗/任务状态
- 深色机能外套，面料防水防磨，多口袋
- 头发扎紧，无碎发
- 随身携带小型感知装置，金属质感，有使用磨损

### 情绪爆发时（关键剧情节点）
- 马尾散开，这是她唯一一次外形"失控"
- 对应她做出某个无法收回的选择的时刻

---

## 性格核心
**表层：** 冷静、高效、怀疑一切。

**内核：** 她见过末世里人性最坏的一面，所以不期待——直到金独字出现，他是第一个让她觉得"信任值得还回去"的人。

**她的独特性：**
金独字没有读过她的完整剧情，所以在她面前他是真实的。她是四人中唯一让他"不知道答案"的人。

**一句话性格总结：**
> 「她学会了不相信任何人——然后遇到了一个值得相信的人，这反而让她更难受。」

---

## 战斗风格
**能力：信息域感知**
- 感知范围内所有生物的位置和意图可见
- 灵笼质感要求：感知展开时不是蓝色光圈，是**空气中细微的波动**，像水面涟漪在三维空间中扩散，肉眼几乎看不见，只有极端特写才能看到
- 代价：感知范围内所有人的情绪她都能感受到——敌人的恐惧和痛苦一并涌入
