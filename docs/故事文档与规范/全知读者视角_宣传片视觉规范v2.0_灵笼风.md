# 《全知读者的视角》系列漫剧
## 序章宣传片 · 视觉规范文档 v2.0
**画风基准：《灵笼》写实厚涂**
*本文档与「人物设定卡 v2.0」配合使用*

## 一、整体视觉基调

### 核心关键词
> **「末世纪实 × 孤独读者 × 命运重量」**

灵笼风格下的宣传片不追求"燃"，追求"重"——
每一帧都让观众感受到这个世界真实存在，
这些人真实地活着，真实地在那个末日里做出了选择。

### 与《咒术回战》风格的根本差异

| 维度 | 咒术回战版（已废弃） | 灵笼版（当前基准） |
|------|---------------------|------------------|
| 线条 | 手绘速写，粗细变化 | 精细均匀，写实边缘 |
| 色彩处理 | 黑白+局部爆色 | 全画面低饱和冷色调 |
| 光影 | 平面漫画阴影 | 体积光+大气散射 |
| 质感 | 概念化 | 皮肤/布料/金属真实质感 |
| 人物比例 | 二次元夸张 | 写实接近概念艺术 |
| 情绪传达 | 表情夸张+大眼高光 | 微表情+动作细节+光影 |
| 技能特效 | 平面光效 | 物理性的力量感+环境反应 |

## 二、色彩与光影规范

### 2.1 全片色调基准

**主色调：冷蓝灰**
- 所有场景的环境光以冷蓝灰为基底
- 饱和度控制在20%–40%之间（参考灵笼整体色感）
- 禁止出现高饱和度的大面积颜色

**对比色逻辑：在统一的冷色调中，用克制的暖色制造情绪焦点**
- 金独字觉醒：暖金色，局部，不超过画面20%面积
- 李洪俊火焰：橙红，局部，作为画面中最"热"的存在
- 系统UI：纯白冷光，带轻微蓝调

### 2.2 三段式光影对应

**第一段「坠落」0:00–0:40 · 压抑光**
- 色调：冷灰，低对比度，画面整体偏暗
- 光源：扩散型天光，无明确方向，像阴天的末日
- 大气效果：轻微雾霾感，远景物体有空气透视虚化
- 人物：面部大面积阴影，眼神是画面最亮的点
- 参考：灵笼第一集登场场景的环境氛围

**第二段「独字」0:40–1:20 · 聚焦光**
- 色调：冷灰底色中开始出现暖色点
- 核心光源设计：**手机屏幕的白光打在金独字脸上**
  - 这是全片最重要的单一光源设计
  - 屏幕光应该有真实的衰减——脸部最亮，向外快速减弱
  - 冷白屏幕光与冷灰环境形成统一，但足够亮来聚焦
- 流中赫登场光：逆光剪影+皮下星纹透光，银白色
- 情绪光变化：从纯冷到冷中带一点暖——代表希望开始出现

**第三段「燃爆」1:20–2:00 · 爆发光**
- 色调：维持冷底色，但四种技能光色同时激活
- 光源叠加顺序：金（独字）→ 橙红（洪俊）→ 银白（流中赫）→ 冰蓝（希珠）
- 爆发光的灵笼处理方式：**不是卡通光爆，是能量密度造成的物理效果**
  - 强光处周围空气有热扭曲/密度扭曲
  - 地面/墙面有光源的真实反射
  - 光边缘有自然衰减，无硬边
- 片名落版：回归冷蓝灰，干净收尾

### 2.3 各角色专属光效（灵笼质感版）

| 角色 | 光色 | 质感要求 | 禁止效果 |
|------|------|---------|---------|
| 金独字 | 暖金 | 从皮下透出，有热度感，边缘模糊 | 卡通发光纹身 |
| 流中赫 | 冷银白 | 皮下燃烧感，物理性地压陷地面 | 平面星座光圈 |
| 李洪俊 | 深橙红 | 真实火焰，有烟有余烬，空气热扭曲 | 卡通火焰效果 |
| 郑希珠 | 冰蓝 | 空气中极细微的波动涟漪，几乎不可见 | 蓝色光环 |
| 系统UI | 冷白蓝 | 半透明界面，边缘轻微发光 | 过于科幻的全息效果 |

## 三、构图规范

### 3.1 灵笼风格构图原则

**核心原则：真实的视角，不是漫画的视角**
灵笼的镜头语言更接近电影，而不是漫画分格。

**宣传片应使用的构图：**
- 电影感宽画幅（16:9或2.39:1超宽画幅）
- 长焦压缩感（人物与背景的距离感被压缩，末世感更强）
- 三分法构图（人物不在正中，留出环境叙事空间）
- 景深效果（前景虚化/背景虚化，强调层次）

**宣传片禁止的构图：**
- 漫画分格式的构图（太平面）
- 过于极端的仰角（灵笼风格下力量感靠质感传达，不靠视角夸张）
- 无景深的全清晰画面（失去电影感）

### 3.2 四角色登场镜头构图规范

**金独字 · 登场镜头（镜头06）**
- 构图：中景，三分法，人物偏左，右侧留出废墟城市
- 焦点：眼神和手机屏幕，背景轻微虚化
- 光源：手机屏幕冷白光打脸，左侧有微弱环境光补光
- 景深：浅景深，背景城市虚化但可辨认
- 禁止：居中构图，全清晰

**流中赫 · 登场镜头（镜头08）**
- 构图：中远景，人物居中但逆光剪影，占画面约1/3高度
- 焦点：轮廓清晰，正面细节由逆光决定（只有发光的纹路可见）
- 光源：背后强逆光（银白色），将他变为有细节的剪影
- 景深：中景深，他清晰，远近都有虚化
- 特别要求：脚下地面有轻微物理压陷的真实感

**李洪俊 · 登场镜头（镜头10混剪）**
- 构图：近景，人物占画面70%，拳头在最前景略虚化
- 焦点：面部表情，特别是眼神
- 光源：火焰从下方打光（橙红底光），配合上方冷环境光
- 景深：拳头前景虚化，面部最清晰

**郑希珠 · 登场镜头（镜头10混剪）**
- 构图：中景侧脸45度，人物偏右，左侧留出感知扫描的空间
- 焦点：眼睛，特别是瞳孔的冰蓝光
- 光源：冷蓝环境光为主，侧脸有轻微补光
- 景深：感知涟漪效果需要中景深来体现空气中的微妙扭曲

## 四、角色宣传片专用POSE库（灵笼写实版）

*灵笼风格下，所有pose的核心是**真实的身体力学**，不是摆好看的造型*

### 金独字 · 宣传片核心POSE

**POSE A「读者凝视」（镜头06）**
- 动作：自然站立，重心略偏，手持手机在腰部高度
- 关键：低头看手机，然后慢慢抬眼——这个抬眼的瞬间是捕捉点
- 真实感要求：不是摆拍，是被抓拍的自然状态
- 光效：手机屏幕光在脸部有真实的方向性投影

**POSE B「文字觉醒」（镜头10）**
- 动作：单膝跪地，一手撑在破碎地面，头低垂，金色文字从掌心蔓延
- 关键：低垂的头，力量是向内收的，然后缓缓抬起
- 真实感要求：地面有被力量压碎的痕迹，手掌有真实的受力感

### 流中赫 · 宣传片核心POSE

**POSE A「降临」（镜头08）**
- 动作：从高处落地的瞬间，单膝微屈缓冲，长发和衣摆因下落惯性飘起
- 关键：落地那一刻脚下地面的物理反应——裂缝、尘土
- 真实感要求：衣摆飘动有真实的物理感，不是漫画式飘逸

**POSE B「作者权限」（镜头10高潮）**
- 动作：直立，双臂微微张开，十二星座纹路在全身皮下激活
- 关键：表情是四人中最复杂的——不是愤怒，是某种沉重的决意
- 真实感要求：皮下发光需要有体积感，皮肤表面有轻微的热变形

### 李洪俊 · 宣传片核心POSE

**POSE A「点火」（镜头10）**
- 动作：握紧双拳，低头，绷带上火焰从指缝渗出
- 关键：肩膀和背部肌肉的真实收紧感
- 真实感要求：绷带边缘有真实的焦化效果，不是卡通火焰

**POSE B「全力冲锋」（镜头10燃爆段）**
- 动作：重心前倾，左脚蹬地，右拳向前推出
- 关键：脚蹬地的那一刻地面反应，拳风有空气压缩效果
- 真实感要求：运动模糊需要符合真实物理，不是放射状速度线

### 郑希珠 · 宣传片核心POSE

**POSE A「扫描」（镜头10）**
- 动作：侧身，一只手轻触左耳耳环，眼睛微微眯起
- 关键：这是一个极度克制的动作——她在全力感知，但外表几乎看不出来
- 真实感要求：感知效果通过极细微的空气扭曲表现，不是发光光环

**POSE B「决断」（镜头11）**
- 动作：马尾散开，正面站立，表情从克制破防的瞬间
- 关键：这是她唯一一次破防——眼眶微红，但没有落泪，嘴唇抿紧
- 真实感要求：散落的发丝有真实的飘动感和发丝质感

## 五、场景视觉规范（灵笼质感版）

### 5.1 三类核心场景设定

**场景A · 末日都市街头**
- 参考：《灵笼》中的巢城外部废土场景质感
- 建筑：现代都市建筑的废墟化，钢筋外露，玻璃碎裂，有真实的破坏逻辑
- 大气：雾霾感，能见度有限，远景物体轮廓模糊
- 地面：积尘、碎玻璃、散落的日常物品——生活在这里戛然而止的证明
- 系统UI元素：漂浮在空中，但用半透明玻璃质感处理，有真实的折射
- 禁止：过于干净的废墟，缺少生活痕迹的场景
- AI绘图关键词：
```
post-apocalyptic Seoul street, realistic destroyed architecture exposed rebar 
shattered glass, heavy atmospheric haze reduced visibility, debris and 
abandoned everyday objects on ground, cold blue-gray overcast light, 
semi-transparent holographic system notification floating in air with glass refraction,
Ling Cage MAPPA animation background quality, extreme detail environmental storytelling
```

**场景B · 系统空间/剧情节点**
- 这是全片最抽象的场景，但用灵笼风格处理要有物理重量感
- 地面：不是平整的光面，是无数发光文字碎片堆积形成的"地面"，有真实的高低起伏
- 背景：不是纯黑，是深空感的极暗蓝黑，有极远处的光源
- 文字：不是平面字符，是有厚度的实体，像玻璃碎片或石板碎片，表面刻着文字
- 光源：文字碎片本身发出暗金色光，是场景唯一光源
- AI绘图关键词：
```
metaphysical void space, ground composed of physical text fragment shards 
with depth and volume like glass or stone pieces, dark blue-black deep space background 
with distant light source, text shards glowing warm amber from within,
single human figure standing, volumetric light from below, 
cinematic dramatic scale, photorealistic rendering, Ling Cage quality
```

**场景C · 战场/技能爆发点**
- 参考：灵笼中的高强度战斗场景，力量感通过环境破坏体现
- 破坏逻辑：每种技能的破坏方式不同——
  - 金独字：文字嵌入地面/墙面，像被刻上去
  - 流中赫：星图物理压陷地面，半径内石块向外推散
  - 李洪俊：爆炸冲击波，地面焦化，高温龟裂
  - 郑希珠：几乎无物理破坏，只有极细微的空气扭曲痕迹
- AI绘图关键词：
```
intense battle aftermath environment, physically realistic destruction matching 
each character's ability type, layered scene with foreground debris midground 
fighters background structural collapse, multiple light sources from different 
ability colors interacting realistically, Ling Cage animation battle scene quality,
photorealistic rendering with cinematic composition
```

### 5.2 灵笼风格场景细节清单

每个场景画面必须包含至少3项以下细节，才能达到灵笼质感：

- [ ] 地面有积灰/积水/碎玻璃等具体物质
- [ ] 远景有大气透视虚化
- [ ] 光源有真实的方向性投影
- [ ] 材质表面有磨损/腐蚀/使用痕迹
- [ ] 空气中有尘埃粒子或雾霾
- [ ] 破坏场景有合理的物理逻辑（爆炸裂纹从爆心向外扩散等）
- [ ] 人物在场景中投下真实的阴影

## 六、系统UI视觉规范（灵笼质感版）

系统通知框是全片最重要的视觉语言，需要在写实风格下保持辨识度。

### UI设计规范
- 材质：**半透明磨砂玻璃质感**，不是科幻全息投影
- 边框：细线，有轻微的折射和玻璃厚度感
- 背景：玻璃磨砂半透明，可以隐约看到后面的场景
- 文字：等宽字体，白色，有轻微发光但不刺眼
- 出现方式：从模糊到清晰，像玻璃从雾气中显现
- 消失方式：整体碎裂成玻璃碎片飘散

### 三次出现内容
| 出现时间 | 内容 | 情绪功能 |
|---------|------|---------|
| 镜头02 · 0:07 | 【剧情已开始 / 시나리오가 시작되었습니다】 | 世界观建立，冷静的震撼 |
| 镜头11 · 1:42 | 手机屏幕【THE END】 | 绝境，但这个人没有放弃 |
| 镜头13 · 1:57 | 【读者，登场了 / 독자가 등장했습니다】 | 点题收尾 |

## 八、文档使用流程

```
制作任意宣传片镜头的流程：

1. 打开宣传片脚本 v1.0，确认镜头编号和内容
2. 打开人物设定卡 v2.0，提取角色基础信息和prompt
3. 在本文档查找：
   ├─ 该时间段的光影规范（第二章）
   ├─ 该角色的构图规范（第三章）
   ├─ 该角色的POSE（第四章）
   ├─ 该场景类型（第五章）
   └─ 是否有UI元素（第六章）
4. 将第七章通用前缀 + 角色设定卡prompt + 本文档规范 组合
5. 生成画面
6. 对照第五章「场景细节清单」验收，不足3项则重新生成
```
